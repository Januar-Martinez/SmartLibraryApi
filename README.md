# API — SmartLibrary

API REST para gestionar prestamos de libros. Construida con **Node.js**, **Express** y **SQL Server Express**.

---

## 🛠️ Tecnologías

- Node.js
- Express
- SQL Server Express (Windows Authentication)
- mssql + msnodesqlv8
- dotenv / nodemon

---

### Crear la base de datos

Ejecuta el script `src/models/schema.sql` en SQL Server Management Studio.
